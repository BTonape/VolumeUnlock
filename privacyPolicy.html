Privacy Policy for Volume Unlock

Balachandra Tonape (a sole developer) built the Volume Unlock application (the "Service").

Permissions Used by the application

  • Device Administrator Permission

    • Volume Unlock is utility app for devices with power button not working, it does both lock and unlock of the device. This app uses the Device Administrator permission to force lock the device from the application, when user agrees for admin permissions. However if the user doesn't want to use the app and wants to uninstall the app, app provides uninstall button which removes the admin permissions and uninstalls the Application.

  • Receive Boot Completed

    • Used to restart the Volume Unlock features when the device is restarted, without opening the application

  • Internet

    • As the application has ads, to load the ads application uses Internet.

Information Collection and Use

Volume Unlock Application do not collect any user information. User Information is private.

Advertisement used

The ads used are from Google Ads AdMob network and are in policy compliance from AdMob.
